Task 2:

Creating a blood pressure data frame using the vectors we created in task 1. In my code for this section, I flip the sign to now use pre-post. This makes it so that that the diff_bp column can correctly return the negative differences in the context of pre-post, rather than post-pre. Note that the data is subset to only include negative differences (lowered blood pressure), and a 5th column is created to determine if the post blood pressure is below 120. Knitr is used to output a nice clean table.

bp_df <- data.frame(c(1:20), pre_treat, post_treat, -(change_BP))
names(bp_df) = c("patient", "pre_bp", "post_bp", "diff_bp")
bp_df <- bp_df[bp_df$diff_bp < 0,]
bp_df$post120 <- (bp_df$post_bp < 120)
